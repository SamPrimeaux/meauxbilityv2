# MeauxResearch Tenant

Research and discovery platform.

## Configuration

- **Worker Name**: `meauxresearch`
- **Subdomain**: `meauxresearch.meauxbility.workers.dev`
- **Custom Domain**: `meauxresearch.dev` (optional)
- **D1 Database**: `meaux-work-db`
- **R2 Bucket**: `inneranimalmedia` (shared)

## UI

Embedded research platform UI (can be customized).

## Deployment

```bash
npm run deploy:meauxresearch
```
