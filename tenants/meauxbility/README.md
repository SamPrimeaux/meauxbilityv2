# Meauxbility Tenant

Main Meauxbility platform tenant.

## Configuration

- **Worker Name**: `meauxbility`
- **Subdomain**: `meauxbility.meauxbility.workers.dev`
- **Custom Domain**: `meauxbility.dev` (optional)
- **D1 Database**: `meauxbility-dashboard-db`
- **R2 Bucket**: `inneranimalmedia` (shared)

## UI

Serves `MEAUXGLOBE.html` and dashboard pages from R2.

## Deployment

```bash
npm run deploy:meauxbility
```
