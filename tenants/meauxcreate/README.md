# MeauxCreate Tenant

Creative platform for content creation.

## Configuration

- **Worker Name**: `meauxcreate`
- **Subdomain**: `meauxcreate.meauxbility.workers.dev`
- **Custom Domain**: `meauxcreate.dev` (optional)
- **D1 Database**: `meaux-work-db`
- **R2 Bucket**: `inneranimalmedia` (shared)

## UI

Embedded creative platform UI (can be customized).

## Deployment

```bash
npm run deploy:meauxcreate
```
