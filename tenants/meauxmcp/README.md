# MeauxMCP Tenant

Model Context Protocol platform.

## Configuration

- **Worker Name**: `meauxmcp`
- **Subdomain**: `meauxmcp.meauxbility.workers.dev`
- **Custom Domain**: `meauxmcp.dev` (optional)
- **D1 Database**: `meaux-work-db`
- **R2 Bucket**: `inneranimalmedia` (shared)

## UI

Serves MCP-related UI from R2.

## Deployment

```bash
npm run deploy:meauxmcp
```
