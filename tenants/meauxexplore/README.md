# MeauxExplore Tenant

Exploration and discovery platform.

## Configuration

- **Worker Name**: `meauxexplore`
- **Subdomain**: `meauxexplore.meauxbility.workers.dev`
- **Custom Domain**: `meauxexplore.dev` (optional)
- **D1 Database**: `meaux-work-db`
- **R2 Bucket**: `inneranimalmedia` (shared)

## UI

Embedded exploration platform UI (can be customized).

## Deployment

```bash
npm run deploy:meauxexplore
```
