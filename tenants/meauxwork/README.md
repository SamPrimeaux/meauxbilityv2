# MeauxWork Tenant

Project management and workflows platform.

## Configuration

- **Worker Name**: `meauxwork`
- **Subdomain**: `meauxwork.meauxbility.workers.dev`
- **Custom Domain**: `meauxwork.dev` (optional)
- **D1 Database**: `meaux-work-db`
- **R2 Bucket**: `inneranimalmedia` (shared)

## UI

Serves `meauxmcp.html` and work-related pages from R2.

## Deployment

```bash
npm run deploy:meauxwork
```
